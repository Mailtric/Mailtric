## WordPress email logging and deliverability insights
>
> Take control of WordPress email reliability. Mailtric logs every email sent via `wp_mail()`, adds one-click resend, and provides SMTP and deliverability diagnostics so you can confirm important emails reach the inbox.

---

## About

Welcome to the official GitHub repository for **Mailtric**. This free WordPress plugin focuses on reliable email delivery: full logging, clear delivery outcomes, one-click resend, SMTP options, and deliverability checks. It works with WooCommerce, form plugins, membership sites, and any plugin using `wp_mail()`. Light and dark mode UIs keep the experience feeling native to WordPress.

**Website:** <a href="https://www.mailtric.com/" target="_blank" rel="noopener noreferrer">www.mailtric.com</a>

---

## Features

- Full email logging - capture subject, recipient, headers, and status for every `wp_mail()` message
- Deliverability insights - understand send outcomes and failures from the log
- One-click resend - resend failed or bounced emails directly from the log
- Clear analytics - overview cards and charts for recent email activity and outcomes
- Smart filters and search - find emails by subject, recipient, date range, or status
- Light and dark mode UI - modern interface that matches your WP theme preference
- Works with your stack - WooCommerce, form plugins, membership plugins, SMTP or transactional providers
- Performance-friendly - efficient logging to avoid slowing down your site
- Simple UI - approachable for clients and non-technical users
- Multilingual admin - English, Spanish, and French interface
- All features free - no pro version, no feature locking, no usage limits
